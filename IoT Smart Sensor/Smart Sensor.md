# [Study] IoT 시스템: 통신, 센서 및 액추에이터 제어

## 1. IoT (사물인터넷) 통신 방식 
* **개요**: 센서 정보를 전달하거나 액추에이터(모터 등)를 동작시키기 위한 네트워크 통신 기술.

### 1.1 주요 통신 프로토콜 비교
| 구분 | 목적 및 통신 대상 | 특징 및 스펙 |
| :--- | :--- | :--- |
| **BLE (Bluetooth)** | 센서 노드 $\leftrightarrow$ **스마트폰** | 저전력 블루투스 통신, 모바일 기기와의 직접 연동에 주로 사용. |
| **ZigBee (지그비)** | 센서 노드 $\leftrightarrow$ **게이트웨이** | - 근거리에서 **250kbps** 속도로 데이터 전송.<br>- 하나의 무선 네트워크에 **16비트 주소** 사용 (최대 65,536개 기기 연결).<br>- 저전력 설계로 적은 배터리로 오래 동작 가능. |

## 2. IoT 주요 센서 (Sensors)
> **아키텍트의 조언**: 하드웨어 제어 시 각 센서가 아날로그(ADC) 통신인지, 디지털(I2C, GPIO Interrupt) 통신인지 파악하는 것이 가장 우선입니다.

### 2.1 환경 및 물리 감지 센서
* **사운드 센서 (Sound)**
    * **원리**: 물체의 진동이나 기체의 흐름에 의해 발생하는 파동 감지.
    * **소리 3요소**: 세기(진폭에 비례), 높이, 맵시(음색).
    * **핀 정보**: BLE 모듈 `G2` (Sound Interrupt Pin), 센서 `INT` 핀.
* **인체 감지 센서 (PIR)**
    * **원리**: 적외선 센서의 일종. 인간/동물에서 나오는 **따뜻한 적외선 양의 변화**를 측정.
    * **핀 정보**: BLE 모듈 `G3` (PIR Interrupt Pin), 센서 `INT` 핀.
* **가스 센서 (Gas)**
    * **원리**: 가스의 누설, 농도, 종류를 판별.
    * **핀 정보**: BLE 모듈 `G2` (ADC Pin), 센서 `GAS_OUT` 핀.
* **온도/습도/조도 센서**
    * **CdS (광센서)**: 주변 빛의 세기에 따라 저항(출력 전압)이 변하는 **아날로그 센서**.
    * **SHT20**: **I2C 통신**으로 주변 온도와 습도 정보를 출력하는 **디지털 센서**.

### 2.2 생체 및 수분/오염 감지 센서
* **펄스 센서 (Pulse / 심박)**
    * **원리**: 적외선 LED 방출 후, 혈류량 변화(원문: 혈관 압력의 미세한 변화)에 따른 적외선 반사/수신량 차이로 심장 박동 추출.
    * **한계**: 맥박만 측정 가능 (혈액 속 산소포화도 측정 불가).
* **Rain/Steam 센서 (수분 감지)**
    * **원리**: 물방울이나 수증기 증가 시 전류가 흘러 `OUT` 단자에 전기 신호(On/Off)를 보내는 모듈.
* **Soil Moisture 센서 (토양 습도)**
    * **원리**: 2개의 다리를 토양에 꽂아, 두 다리 사이에 흐르는 **전류량**으로 습도 측정.
* **Dust 센서 (광학 먼지 센서)**
    * **원리**: 적외선 발광 $\rightarrow$ 먼지/연기 입자에 부딪혀 반사 $\rightarrow$ PD(Photo Diode) 부분에 수광되는 양으로 공기 오염도 측정.
    * **측정 주기**: 10ms.

### 2.3 위치, 동작 및 거리 측정 센서
* **TILT 센서 (기울기 스위치)**
    * **원리**: 각도를 정밀하게 측정하는 것이 아닌, 일정 각도 이상 기울어지면 **On/Off** 되는 스위치 기반 센서.
    * **출력**: 센서 On 시 연결된 핀(예: HBE-ADK-2560)에 `LOW` 값 출력.
* **Ultrasonic 센서 (초음파 거리)**
    * **원리**: 초음파를 발사하여 물체에 부딪혀 돌아온 **시간(Time-of-flight)**을 측정해 거리(cm) 계산.
    * **단위**: 프로그램에서 거리는 `cm`, 시간은 `microseconds` 사용. *(주의: 왕복 시간이므로 계산 시 /2 필수)*
* **각속도 센서 (GYRO Sensor)**
    * **원리**: 팽이의 관성을 이용하여 회전하는 방향(각속도)을 측정하는 센서.

## 3. 액추에이터: DC 모터 (DC Motor)

### 3.1 DC 모터의 개요 및 특징
* **구조**: 고정자(영구자석) + 회전자(전기자/로터, 코일) + 브러시 + 베어링 + 모터 케이스.
* **원리**: 전기자에 흐르는 전류의 방향을 전환하여 자력의 반발력과 흡인력으로 회전력을 생성. (주로 로봇 바퀴에 사용)
* **특징**:
    1.  기동 토크(초기 회전력)가 크다.
    2.  인가 **전압**에 대해 회전 특성(속도)이 직선적으로 비례.
    3.  입력 **전류**에 대해 출력 토크가 직선적으로 비례 (효율 양호).
    4.  제어 회로가 단순하고 제어가 쉬우며 가격이 저렴함.

### 3.2 H 브리지 (H-Bridge) 제어 회로 

[Image of H-bridge circuit for DC motor control]

* **정의**: 단일 전원으로 모터에 가하는 **전압의 방향**을 바꿀 수 있는 스위칭 회로.
* **동작 원리 (방향 제어)**:
    * **정회전**: `Q1`, `Q4` 트랜지스터만 동시 On (Q1 $\rightarrow$ Q4 쪽으로 전류 흐름).
    * **역회전**: `Q2`, `Q3` 트랜지스터만 동시 On (Q2 $\rightarrow$ Q3 방향으로 전류 흐름).

### 3.3 모터 속도 제어 (Speed Control) 
> *구동 전압을 변화시켜 속도를 제어하는 2가지 방법*

1.  **아날로그 방식**
    * 직접 구동 전압의 크기를 연속적으로 변화시키는 방식.
2.  **펄스폭 변조 방식 (PWM: Pulse Width Modulation)**
    * **원리**: 모터 구동 전원을 일정 주기로 **On/Off 하는 펄스 형상**으로 공급 (디지털 제어).
    * **Duty 비 (Duty Cycle)**: 펄스 1주기 내에서 `On 시간`과 `Off 시간`의 비율을 바꿈으로써 평균 전압을 조절하여 속도를 제어함.
    * **특징**: DC 모터가 매우 빠른 주파수의 변화에는 즉각적으로 기계적 반응을 하지 못하는 관성(Inertia) 특성을 이용함.
