# 📝 [기술 노트] 실무 핵심

## 1. 개발 도구 및 환경 (Vi 편집기)
### 모드 전환
- **`i`**: 명령 모드 → **입력 모드** (Insert)
- **`ESC`**: 입력 모드 → **명령 모드**

### 유용한 단축키
- **`A`**: 현재 줄의 **맨 끝으로 이동**하여 입력 모드 전환 (매우 유용)

## 2. 프로그래밍 철학 및 코딩 습관
### 코드 품질의 기준
- 코드의 길이는 중요하지 않다. **복잡도(Complexity)를 낮추는 것이 핵심**이다.
- 함수는 **작게, 기능 단위**로 쪼갤 것.

### 작성 순서 및 습관
- **주석 먼저 쓰기**: 코드를 짜기 전에 주석으로 로직을 먼저 설계하라.
- **메모 습관**: 생각나는 아이디어나 로직은 반드시 적어두는 습관을 들일 것.
